## Goal

The goal of this exercise is to teach the student about classes in Kotlin.

## Learning objectives

- Know how to write a class
- Know how make instances of a class
- Know about primary and secondary constructors
- Know about `init` blocks

## Out of scope

- Inheritance
- Data classes
- Extensions

## Concepts

`classes`:
- know how to write a class
- know about primary and secondary constructors
- know about `init` blocks

## Prerequisites

- conditionals
