# Design

## Goal

The goal of this exercise is to teach students about higher order functions and lambda expressions in Kotlin.

## Learning objectives

- How to write lambda expressions
- How to pass a function to another function
- How to call a function from a variable or function parameter

## Out of scope

## Concepts

- `higher-order-functions`: Know about lambda expressions in Kotlin and how to assign them to variables; how to pass them to other functions; how to return them; how to invoke them 

## Prerequisites

Strings and numbers are used in the exercise.
